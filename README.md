📊 Bank Loan Data Analysis Project
📌 Project Overview

This project analyzes bank loan data (dummy dataset) to understand the status and performance of loans across different client segments.

The goal is to demonstrate practical skills in:

Data modeling and querying using SQL Server

Data visualization and dashboard creation using Power BI

Business-oriented analysis of financial data

This project simulates a real-world banking scenario where decision-makers need insights into loan approval, repayment behavior, and risk distribution.

🛠️ Tools & Technologies

SQL Server – Data storage, cleaning, and querying

Power BI – Dashboard creation and data visualization

T-SQL – Data aggregation and business logic implementation

📂 Dataset Description

The dataset contains dummy information about:

Client details (age, income, employment status, etc.)

Loan details (loan amount, type, term, interest rate)

Loan status (Current,active, closed)

The data was structured into relational tables inside SQL Server before being connected to Power BI.

📊 Power BI Dashboard

An interactive dashboard was built in Power BI to visualize:

Total loans applcations and total funded & received amount

Loan status distribution

Default analysis

Loan breakdown by:

Client seniority

Loan type

Term length

The dashboard allows filtering and slicing by loan purpose, loan type, location and others to simulate real business analysis.

📈 Key Business Insights

Identification of high-risk loan segments

Comparison between bad and good applications

Evolution of loan applications

🎯 Project Objective

This project demonstrates the ability to:

Work with relational databases

Write optimized SQL queries for business analysis

Transform raw data into actionable insights

Build clear and professional BI dashboards
